Node.js est un framework libre événementiel en JavaScript orienté vers les applications réseau qui doivent pouvoir monter en charge. Il utilise la machine virtuelle V8 et implémente sous licence MIT les spécifications CommonJS.

Node.js contient une bibliothèque de serveur HTTP intégrée, ce qui rend possible de faire tourner un serveur web sans avoir besoin d'un logiciel externe.
